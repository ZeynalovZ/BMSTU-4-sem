Интерполяция полиномом Ньютона

y(x) = y0 + (x-x0)y(x0, x1) + (x-x0)(x-x1)y(x0, x1, x2) + ... + (x-x0)(x-x1)...(x-xn+1)y(x0, x1, ..., xn)

Где y(x0, x1) - разделенная разность

y(x0, x1) = (y0 - y1)/(x0 - x1)

y(x0, x1, xn) = (y(x0, x1, ..., xn-1) - y(x1, x2, ..., xn))/ (x0 - xn)

1) вход - таблица данных х-у

Заполняется самостоятельно (например у = 4x^2) и загружается в программу из файла

2) вход - степень полинома n

используется n+1 узлов (то есть точек, строк таблицы, уравнений)

3) вход - значение х

Ищем х в таблице и выбираем n+1 близжайших точек.

если х нет в таблице - происходит экстрополяция. Выдать об этом сообщение и посчитать с помощью доступных близжайших значениях

4) вывод - y(x) = res. FuncRes
